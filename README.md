 Website Traffic Analysis

 Objective
 
Analyze website traffic data and identify key user engagement trends.

 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- GitHub

 KPIs
- Total Users
- Total Sessions
- Total Engaged Sessions
- Average Engagement Rate

 Visualizations
1. Traffic Source Analysis
2. Users by Traffic Source
3. Engagement Rate Analysis

 Key Insights
- Organic Social generated the highest traffic.
- Direct and Organic Search channels contributed significantly.
- Engagement varied across traffic sources.
- Social media was the primary acquisition channel.

## Outcome
Built an interactive dashboard to analyze website performance and user engagement.
